# MCopy

A lightweight macOS menu bar clipboard manager built with SwiftUI + SwiftData.

## Features

- Menu bar app with clipboard history panel
- Global hotkey to open history (`⌘⇧V` by default)
- Search clipboard history and paste selected item quickly
- Configurable panel position on screen (`Left / Bottom / Right / Top`)
- Settings window with shortcut customization

## Shortcuts

- `⌘⇧V`: Show history panel
- `⌘,`: Open Settings
- `⌘Q`: Quit MCopy

## Run

1. Open `MCopy.xcodeproj` in Xcode
2. Select the `MCopy` scheme
3. Build and run
